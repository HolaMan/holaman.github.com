Button 按鈕與Click事件

了解如何使用Button (按鈕)

說明

Button是最常用的視覺元件 (View)。 Button最主要的功能就是讓使用者去按下它，按下它會產生一個 觸發事件 (稱為Click Event)。 所以我們去監聽Click Event，當使用者按下它了，我們就可以針對這個事 件作對應的處理。 收聽Click有二種作法，一種直接定義在XML裡，另一種則是寫程式去監聽。

程式範例

方法#1

用程式監聽Button的Click事件
public class ToastActivity extends Activity {
    @Override
    public void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_toast);        
        
        Button button4 = (Button)this.findViewById(R.id.button4);
        button4.setOnClickListener(new OnClickListener() {
            @Override
            public void onClick(View v) {
                Toast.makeText( ToastActivity.this, "按鈕被按了....", Toast.LENGTH_LONG).show();
            }
        });
    }
}

方法#2

在XML定義收聽Button的Click事件的對應函式
<RelativeLayout>    
    <Button
        android:id="@+id/button4"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:onClick="showDialog"
        android:text="Button Click" />
</RelativeLayout>
當然在Activity的程式碼加上以上的函式即可
public void showDialog(View view) {
    // do something
}
